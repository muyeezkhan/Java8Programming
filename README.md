# Java 8 Practice Programs

This repository contains a collection of Java 8 programs demonstrating core language concepts and features introduced in Java 8. It is designed for learners, interview preparation, and anyone looking to strengthen their understanding of modern Java programming.

## Features Demonstrated
- Lambda Expressions
- Functional Interfaces (Predicate, Function, Consumer, Supplier, etc.)
- Stream API (filter, map, reduce, collect)
- Method References
- Optional Class Usage
- Date and Time API (java.time)
- Collections with Streams
- General Java Problem-Solving Examples

## Purpose
- Improve understanding of functional programming in Java
- Practice commonly asked interview coding patterns
- Provide reusable reference code for quick recall
